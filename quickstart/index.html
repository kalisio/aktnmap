# Introduction

Akt'n'Map is a situational intelligence solution to inform and coordinate fieldworkers during events like interventions, crisis management, etc.

Akt'n'Map is available as:

  • a web application on https://aktnmap.kalisio.com,
  • a mobile Android application on Google Play,
  • a mobile iOS application on Apple Store.

Once registered to the application you can access all versions without any limit.
