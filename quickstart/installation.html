# Installation

# Web application

The web application is available on https://app.aktnmap.com.

It has been tested on the following browsers:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

TIP

We advise you to keep your browser up-to-date in order to use a reliable and secure version.

The web application does not require any installation workflow.

# Mobiles applications

You are a fieldworker ? Android and iOS apps can help you monitor and capture events on the go.

Use mobile app notifications to keep tabs on work that needs your attention and take action faster right from the app.

# Android

The Android application is available to install on Google Play

# iOS

The iOS application is available to install on Apple Store.
