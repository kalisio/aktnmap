# A propos

Akt'n'Map est un projet libre disponible sur GitHub.

Il repose sur un écosystème de développement et de déploiement composé de:

  • KDK, un kit de développement pour les applications géospatiales
  • Kano, un visualiseur 2D/3D de données géospatiales
  • Weacast, une plateforme de gestion de données météorologiques,
  • Kargo, une solution pour déployer des services géospatiaux
  • Kaabah, une solution pour déployer et opérer des infrastructures
