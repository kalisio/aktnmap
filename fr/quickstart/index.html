# Présentation

Akt'n'Map est une solution d'intelligence situationnelle pour informer et coordonner les acteurs de terrain lors d'événements tels que des interventions, la gestion d'une crise, etc.

Akt'n'Map est disponible sous la forme d'une:

  • application internet sur https://app.aktnmap.com,
  • application mobile Android sur Google Play,
  • application mobile iOS sur l'Apple Store.

Une fois enregistré sur l'application vous pouvez accéder à toutes les versions sans restriction.
